# ML-CCFD Credit Card Fraud Detection
  本项目旨在围绕不均衡数据(Credit Card Fraud Detection)问题开展实验研究并撰写分析报告。主要任务如下：
1.	探讨类别不均衡对传统分类模型表现的影响；
2.	对比不同重采样技术（如欠采样、过采样等）的应用效果，进行综合分析；
3.	针对SMOTE算法在实际应用中的问题，尝试提出优化思路或创新方法。
